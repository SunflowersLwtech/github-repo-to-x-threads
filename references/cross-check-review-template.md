# Cross-Check Review

Use this file as the required review gate before publishing a thread.

## Review Status

- Status: `draft`

Status options:

- `pass`: ready to post.
- `revise`: needs edits before posting.
- `block`: do not post until evidence or ownership risk is resolved.

## Required Checks

- Every public factual claim maps to a source in `claims_ledger.json`.
- Claims from GitHub metadata are current for this run.
- Claims from README/docs/code include file paths.
- Personal vision is labeled as personal vision.
- Inferences are not phrased as official roadmap or shipped facts.
- Unsupported "best", "first", "only", "most complete", benchmark, production-readiness, or roadmap claims are removed or downgraded.
- The repo owner is credited in the first post.
- The user relationship to the repo is clear.
- Generated images are marked as conceptual unless they are actual repo assets.
- No fake screenshots, fake metrics, fake benchmark charts, or fake official assets are used.
